## Studying Material
### Excel
- `categoryName`: module
	- Academic: "CVS"
	- Clinical: "Cardio"
- `tag` (optional): subject
	- Academic: "Anatomy"
	- Clinical: "CBL", "Tutorials"
- `subCategoryName`: lecture/chapter
	- If the main studying content is PowerPoint, I suggest adding per lecture
		- If you have a lecture "Tryptophan and tyrosine metabolism" add as a single entry (if both in the same PowerPoint)
	- If it's a book, I suggest adding by section
		- Two entries: "Tryptophan" and "Tyrosine"
	- If you have two lectures with the same prefix (eg, Errors of refraction I and Error of refraction II), You should merge them into one to avoid confusing the AI labeling algorithm.

> [!caution]- Common Mistakes
> - NOT `category` but `categoryName`
> - NOT `subCategory` but `subCategoryName`
> - NOT `tags` but `tag`

- `Cas`:
	- If two questions has the exactly the same value, they'll be grouped.
	- For matching questions, I use "Match the following" as the `Cas` and I create a question for each option.
- `Tags` and `Year`:
	- The first tag represents the source and can be set from the settings, while the second gives more information.
	- ![[Image of settings\|Image of settings]]
		- Source priority is determined by order.
	- Examples for sources tags and additional ones for each type:
		- `Exams`
			- Additional: "Final/End/Formative/Practical" + YEAR
		- `Department`: official university Q banks
		- `Professor`: PowerPoint questions of a certain professor
			- Additional: Professor name
		- `Guyton`...
	- Merging duplicates rules:
		- All tags will be merged removing duplicates.
		- If `Year` is different than that of the source with the highest priority will be used.
		- If the questions are from the same source, the most recent `Year` will be used.

> [!caution]- Tags vs tagSuggere
> - `Tags` for sources, eg, "Exams, Final 2025"
> - `tagSuggere` for the subjects, it must be identical to the `tag` in [[Admin Tips#Studying Material\|Admin Tips#Studying Material]]

> [!caution]- EXP vs Exp
> - EXP will be the final explanation displayed to user. Use it if you're adding a text book questions that already have explanations (eg, Guyton).
> - Exp will be local to the admin. The AI will use it to generate the explanations.
